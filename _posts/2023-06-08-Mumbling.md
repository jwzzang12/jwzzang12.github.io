### Mumbling

```javascript
function accum(s) {
  return s
    .split("")
    .map((el, idx) => el.toUpperCase() + el.toLowerCase().repeat(idx))
    .join("-");
}

console.log(accum("AbCde"));
// A-Bb-Ccc-Dddd-Eeeee
```
